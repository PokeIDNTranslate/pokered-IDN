# Pokémon Merah & Biru [![Build Status][ci-badge]][ci]

This is a translation of the Original worldwide releases of Pokemon, Red & Blue to the Indonesian language

It builds the following ROMs:

- Pokemon - Versi Merah (Indonesia) (SGB Enchanced).gb `sha1: **Coming once version 1.0.0 is out**`
- Pokemon - Versi Biru (Indonesia) (SGB Enchanced).gb `sha1: **Coming once version 1.0.0 is out**`
- POKEBIRU.gb (debug build) `sha1: **Coming once version 1.0.0 is out**`
- pokemerah.patch (3DS Virtual Console) `sha1: **Coming once version 1.0.0 is out**`
- pokebiru.patch (3DS Virtual Console) `sha1: **Coming once version 1.0.0 is out**`

To set up the repository, see [**INSTALL.md**](INSTALL.md).
